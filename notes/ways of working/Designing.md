For UI / UX design, unless you want something generic to pop out when you ask for a single page web app, you should give Claude Code some context around what is good design and UX.  
To do this with your own custom sensibilities you would create a bunch of skills and have each one focus on evaluating requirements against, e.g. usability, accessibility, color-science, typography... 
But if you don't have time for that install something like this:

https://github.com/nextlevelbuilder/ui-ux-pro-max-skill

It's looks fancy but basically it's a giant list of stills/agents that someone else already put together.  It will produce much better results from scratch than without it.

### Figma
You can install Figma's MCP to let CC look at your designs, and I think now it can also make changes.  I've had mixed luck with this.
