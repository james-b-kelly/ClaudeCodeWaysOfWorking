
You have a `/notes` folder.  Open Obsidian, click Manage Vaults (bottom left), and choose Open Folder as Vault, selecting your notes folder - the same as you did for this vault.

Claude Code deals mostly in markdown files, and Obsidian makes them nicely readable, and they live-update as they're changed.  You can also link between notes by typing `[[` and choosing an existing file. 

This makes it perfect for reading Claude Code's output, like plans, spikes, reports etc.

You can use the same approach above to have Claude Code manage your notes.

Tell it about your notes folder
```
I keep my notes in an Obsidian vault in `/notes`, note that in your CLAUDE.md.
```
And from now on it will know that when you say "write this up in my notes" it'll know to use this folder.

You can ask CC to organise your notes any way you like (or ask it what the best way would be).
