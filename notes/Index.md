This explains a way of working with Claude Code that allows you to
- Create and refine tickets (by hand, or in conjunction with Claude Code).
	- Have a gated work flow with specific steps so you can keep track of work.
- Keep notes and plans organised automatically by Claude Code.
- Manage context and awareness of current goals across multiple chat/context windows.
- Manage long term memory of important information for your project in context files.
- Have a local relational database of components in a code base for quick reference.
- Have a local vector database for making sense of large unorganised textual data sets (like a big folder of files).
- Create Skills to run common prompt flows, like saving a prompt for a fresh context automatically.
- Create scripts to run common tasks predictably.
